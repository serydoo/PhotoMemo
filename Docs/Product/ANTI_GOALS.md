# Anti Goals

Last updated: 2026-06-23

## Status

```text
Frozen
```

PhotoMemo should never become:

- its own gallery
- its own timeline
- its own map
- its own people manager
- its own search system
- its own photo browser
- its own image editor
- its own dashboard
- its own workspace
- its own task center

These systems belong to Apple Photos.

PhotoMemo only extends memory understanding and memory workflow.
