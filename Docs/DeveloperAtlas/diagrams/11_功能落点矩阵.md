# 功能落点矩阵

| 新功能类型 | 第一入口 | 关键下游 | 主要风险 |
| --- | --- | --- | --- |
| V1 预览模块 | `V1PreviewCompositionEngine` | `V1PreviewSyncCoordinator`, `ConfigurationSession` | display text 和 template token 混淆 |
| Metadata 字段 | `PhotoMetadata`, `PhotoMetadataReader` | `MetadataContext`, `CardVariableProvider` | 低 metadata 图片 fallback |
| Memory Subject 能力 | `MemorySubject` | `MemorySubjectAdapter`, `MemoryExpressionEngine` | V1 兼容路径和 V2 对象语义混在一起 |
| Configuration Center 对象编辑 | `ConfigurationCenteriOSView` | `ConfigurationSession`, `ConfigurationCenterState` | 重开 IA-002 架构 |
| 默认配置字段 | `V1ConfigurationApplyCoordinator` | `ConfigurationCoordinator`, `SettingsService`, `BatchConfigurationSnapshot` | 保存、bootstrap、share snapshot 不一致 |
| 分享接单 | `ExternalPhotoIntakeCenter` | `ShareCoordinator`, `BatchQueueStore` | 重复来源、临时文件、默认配置冻结 |
| 队列执行 | `BatchQueueStore` | `BatchProcessingCoordinator` | 生产状态、恢复、通知 |
| 输出样式 | `TemplatePreset`, `RecordCardRenderer` | `RecordCardExportService` | Renderer 承担过多布局真相 |

## 使用方法

先在左列找到功能类型，再从“第一入口”开始读代码。

如果同一个功能跨两行，优先把问题拆清楚再实现。
