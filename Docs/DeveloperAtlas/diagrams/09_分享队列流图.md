flowchart TD
    A["Apple Photos / Share / Open URL"] --> B["PhotoMemoRootSceneView"]

## 读图方式

- share intake 和 batch queue 是连续链路
- 真正生产处理在 `BatchProcessingCoordinator`
- queue 状态和默认 configuration snapshot 会影响后续输出
