先看：

1. [21_风险分析（哪些地方不能乱改）.md](/Users/rui/Desktop/PhotoMemo/Docs/DeveloperAtlas/21_风险分析（哪些地方不能乱改）.md)

3. [09_代码依赖图.md](/Users/rui/Desktop/PhotoMemo/Docs/DeveloperAtlas/09_代码依赖图.md)

判断顺序：

1. 它是不是入口文件
2. 它是不是共享状态或持久化文件
3. 它是不是生产 pipeline 文件
4. 它是不是 renderer / export / metadata 边界

如果答案里有两个以上是“是”，先做小范围计划再改。
