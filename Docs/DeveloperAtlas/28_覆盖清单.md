## 还没有深挖

这些主题已有入口，但还没有做到逐类型或逐函数的深层分析：

- `PhotoMemoiOSV1View` 内部每个页面 section 的详细拆解
- `SettingsService` 每个持久化 key 的详细说明
- `BatchQueueStore` 执行状态机的逐步分析
- `RecordCardExportService` metadata 写回细节
- `ConfigurationSession` 所有 mutation 方法
- `MemoryExpressionEngine` 策略扩展细节

这些应进入后续深化路线，而不是继续塞进第一版。
