# MemoMark v1.7.0（Build 7）更新说明

日期：2026 年 7 月 17 日

## 本次更新

- 首页新增“为什么开发时光记”模块，放在“记忆对象”上方。
- 用简洁内容说明产品的开发初衷：陪伴孩子成长、从照片中快速看到当时的年龄，并从出生日期扩展到纪念日和未来的重要日期。
- 设置 -> 使用与帮助内新增“表达公式说明”入口，点击后打开独立二级窗口。
- 表达公式按时间锚点分类，依照当前预设顺序展示每种表达的“之前、当时 / 当日、之后”三种情况。
- 主体、智能输出、锚点结果使用不同颜色标识，帮助第一次使用的用户快速理解表达组成。
- 修复首页配置和时间锚点滑动删除可能触发的 iOS 崩溃。删除操作现在只有在确认后才会真正修改数据。

## 真机验证

- `PhotoMemoiOS` arm64 真机构建成功。
- 已安装并启动到 iPhone7，保留原有本地数据。
- 已确认首页新增模块位于“记忆对象”上方，页面层级和文字显示正常。
- 设置入口使用原生 Sheet 二级窗口，支持中等和大尺寸展示。
- 表达公式、首页滑动操作、时间锚点删除确认相关聚焦测试通过。

## 说明

本次更新不包含 BrandMark 研究目录、品牌资源候选文件或其集成计划。原图保留、本地优先和 Apple Photos 工作流保持不变。
