# 时间锚点与表达语气说明

## 一句话理解

时间锚点是一个有意义的日期，例如宝宝生日、结婚日、相识日、考试日或自定义纪念日。时光记用“照片拍摄日期”与这个日期比较，先算出时间结果，再根据选择的表达语气组成一句容易阅读的话。

## 一条表达是怎样生成的

```text
选中的主体 + 选中的时间锚点 + 照片拍摄日期
                    ↓
          计算时间方向与时间结果
                    ↓
              套用表达语气
                    ↓
                最终显示文字
```

本文统一使用：

- 选中的主体：宝宝“小满”
- 宝宝生日锚点：2025 年 1 月 1 日
- 锚点后的示例结果：1岁2个月18天
- 锚点前的示例结果：还有86天（句子中也可能使用不带“还有”的 `86天`）
- 自定义锚点名称：第一次上幼儿园

## 先分清四种可替换内容

| 内容 | 含义 | 示例 |
|---|---|---|
| 主体 | 当前选中的记忆对象 | 小满 |
| 锚点名称 | 当前配置选中的日期名称 | 出生、第一次上幼儿园 |
| 年龄结果 | 生日锚点之后的年龄 | 1岁2个月18天 |
| 时长或倒计时结果 | 其他锚点前后的时间差 | 2年3个月、还有86天、86天 |

智能变量只负责输出结果，不强制生成整句话。例如：

- `{{anchor_age_text}}` → `1岁2个月18天`
- `{{anchor_countdown_text}}` → `还有86天`
- `{{anchor_duration_text}}` → `2年3个月`
- `{{anchor_title}}` → `第一次上幼儿园`

用户可以把文字和变量组合成自己的句子，例如：

```text
今天，小满已经 {{anchor_age_text}}
```

最终得到：

```text
今天，小满已经 1岁2个月18天
```

## 生日锚点

生日锚点会在照片早于生日时表达“等待出生”，在照片晚于生日时表达“拍摄时年龄”。主体名称会参与大部分表达。

| 语气 | 锚点前示例 | 锚点后示例 | 适合感觉 |
|---|---|---|---|
| 自然（默认） | 还有86天，小满就要出生了 | 今天小满1岁2个月18天 | 直接、日常 |
| 仪式感 | 再过86天，就是小满来到世界的日子 | 今天是小满1岁2个月18天 | 郑重、纪念 |
| 成长 | 距离第一次见面还有86天 | 小满长到1岁2个月18天了 | 强调成长 |
| 温馨 | 等待小满到来，还有86天 | 陪小满走到1岁2个月18天 | 陪伴、柔和 |
| 极简 | 小满出生倒计时：86天 | 小满｜1岁2个月18天 | 简洁、克制 |

如果开启周年发生日表达，系统会改为下一次生日，例如：

- 自然：还有86天，就是小满2岁生日
- 成长：小满还有86天迎来2岁
- 温馨：再等86天，小满就2岁了

## 结婚锚点

结婚锚点主要围绕“结婚之前还有多久”或“婚后已经多久”，当前公式通常不插入主体名称。

| 语气 | 锚点前示例 | 锚点后示例 |
|---|---|---|
| 自然（默认） | 结婚还有86天 | 结婚已经2年3个月 |
| 仪式感 | 再过86天，就是结婚的日子 | 今天是婚后2年3个月 |
| 温馨 | 距离结婚还有86天 | 与你相伴2年3个月 |
| 极简 | 结婚倒计时：86天 | 婚后2年3个月 |
| 回忆 | 距离那一天还有86天 | 从那一天起，已有2年3个月 |

周年表达示例：`还有86天，就是结婚3周年`。

## 相识或关系锚点

这类锚点使用用户填写的锚点名称，例如“第一次见面”“成为一家人”。锚点前的自然语气会同时使用主体和锚点名称，其他语气主要使用锚点名称。

| 语气 | 锚点前示例 | 锚点后示例 |
|---|---|---|
| 自然（默认） | 小满距离第一次见面还有86天 | 第一次见面已经2年3个月 |
| 仪式感 | 再过86天，就是第一次见面 | 今天是第一次见面2年3个月 |
| 回忆 | 距离第一次见面还有86天 | 自第一次见面起，已有2年3个月 |
| 温馨 | 期待第一次见面，还有86天 | 关于第一次见面的故事，已有2年3个月 |
| 极简 | 第一次见面倒计时：86天 | 第一次见面｜2年3个月 |

## 考试锚点

考试锚点使用具体考试名称，例如“高考”“期末考试”。它强调倒计时、冲刺或考试之后过去的时间。

| 语气 | 锚点前示例 | 锚点后示例 |
|---|---|---|
| 自然（默认） | 距离高考还有86天 | 高考已经过去2年3个月 |
| 仪式感 | 再过86天，就是高考 | 从高考那天起，已经2年3个月 |
| 激励 | 冲刺高考还有86天 | 高考结束已经2年3个月 |
| 极简 | 倒计时：86天 | 已过2年3个月 |
| 记录 | 高考还有86天 | 自高考以来，已有2年3个月 |

## 自定义锚点

自定义锚点适合入园、搬家、旅行、第一次走路等个人事件。句子主要使用锚点名称。

| 语气 | 锚点前示例 | 锚点后示例 |
|---|---|---|
| 自然（默认） | 距离第一次上幼儿园还有86天 | 自第一次上幼儿园起，已有2年3个月 |
| 仪式感 | 再过86天，就是第一次上幼儿园 | 今天是第一次上幼儿园2年3个月 |
| 回忆 | 距离第一次上幼儿园还有86天 | 从第一次上幼儿园那天起，已有2年3个月 |
| 温馨 | 期待第一次上幼儿园，还有86天 | 关于第一次上幼儿园，已经2年3个月 |
| 极简 | 第一次上幼儿园倒计时：86天 | 第一次上幼儿园｜2年3个月 |

## 给第一次使用者的选择建议

- 不知道怎么选：使用“自然（默认）”。
- 宝宝成长照片：可选“成长”或“温馨”。
- 生日、纪念日封面：可选“仪式感”。
- 图片上已经有较多文字：可选“极简”。
- 想强调过去的故事：可选“回忆”。
- 考试前的记录：可选“激励”。

## 公式阅读方法

看到下面的公式：

```text
等待[主体]到来，还有[倒计时天数]
```

依次替换：

```text
[主体] = 小满
[倒计时天数] = 86天
```

最终得到：

```text
等待小满到来，还有86天
```

如果公式中没有 `[主体]`，就不会因为切换记忆对象而改变称呼；如果包含 `[锚点名称]`，切换当前生效锚点后，句子会使用新锚点的名称和日期重新计算。

## 与预设的关系

预设决定照片卡片如何显示；当前配置决定选中哪个主体、哪个时间锚点和哪种表达语气。照片进入处理时，会使用当时生效的配置计算结果。切换预设或锚点后，需要保存当前配置，后续处理才会稳定使用新的组合。

## 代码事实来源

本说明依据当前实现中的 `MemoryAnchorExpressionStyle`、`MemoryAnchorExpressionResolver` 与模板变量定义整理。示例数字用于帮助理解，不代表固定输出；真实结果由锚点日期、照片拍摄日期和日历计算共同决定。
