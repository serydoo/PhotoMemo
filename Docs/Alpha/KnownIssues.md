# PhotoMemo Alpha Known Issues

Last updated: 2026-06-20

This file tracks issues already understood by the project, even if they are not being fixed immediately.

## Current Known Issues

### KI-001

- Area: Share-first workflow
- Summary: The Share Extension is still more of an intake bridge than a complete primary product experience.
- Impact: Users may still feel that share-to-PhotoMemo is a handoff instead of an immediate generate-and-save flow.
- Status: Known

### KI-002

- Area: iPhone editing
- Summary: Chinese IME behavior has been improved, but still needs longer real-device validation in continuous editing sessions.
- Impact: Heavy template editing on iPhone may still reveal rough edges.
- Status: Needs validation

### KI-003

- Area: Trust and completion feedback
- Summary: Save success and workflow completion signals still need stronger confirmation inside real use.
- Impact: First-time users may be unsure whether the result is already finished or only queued.
- Status: Known
