# PhotoMemo Product Specification

## Product Position

PhotoMemo is a local-first memory card generator.

Not:

- Gallery
- Photo Editor
- AI Generator

Is:

- EXIF Visualizer
- Anniversary Card Generator
- Family Memory Tool

---

## Core Workflow

1. Configure Template

2. Configure Anchors

3. Save Settings

4. Preview Result

5. Select Photo

6. Generate Card

7. Save To Album

---

## Entry Methods

Current:

Manual Photo Selection

Future:

Share Extension

---

## Template Philosophy

Never modify original pixels.

Generate a new image.

Information is rendered in reserved areas.

---

## Current Template

Classic White

Bottom Fixed Info Bar

Three Column Layout

Left:
Story + Time

Center:
Badge

Right:
EXIF + Anniversary
