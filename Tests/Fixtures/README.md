# MemoMark Fixtures

This directory stores long-term regression fixtures.

Current committed assets include:

- `FixtureManifest.json`
- `GenerateSyntheticFixtures.swift`
- `Synthetic/`

The `Synthetic/` directory now contains deterministic project-owned JPEG/HEIC binaries generated by the committed Swift script.

Use [FixtureSpecification.md](/Users/rui/Desktop/PhotoMemo/Docs/FixtureSpecification.md) and `FixtureManifest.json` as the source of truth for:

- committed and reserved filenames
- required metadata characteristics
- regeneration rules
- future fixture onboarding rules
