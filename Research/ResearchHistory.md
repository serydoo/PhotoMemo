# Research History

## 2026-06-22

PhotoMemo entered V2 Research Phase.

Development, UI polishing, and renderer polishing were paused so the project can become a local-first Photo Presentation Engine with research, specification, and Layout Engine as first-class assets.

The Project Constitution was added as the highest-level repository rule. It clarified that old documentation should not be migrated immediately; new research documentation must be built first, and migration should wait until research specifications stabilize.
