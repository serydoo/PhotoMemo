# Research History

## 2026-07-20

MemoMark archived the initial Expression Style System thesis as a possible V4
research direction. The seed separates user-owned Presets, semantic
MemoryBehavior, visual Expression Style, and bounded Style Variants so future
style growth does not become an unstructured template catalog.

The archive records Classic, Minimal, Signature, Film, and Magazine as visual
research candidates, while Baby, Travel, Journal, and Story remain behavior
candidates. A horizontal real-preview Style Library is preserved as an
interaction hypothesis inside the existing Configuration Center boundary.

This work changed documentation only. It did not start V4, freeze a PDR, or
authorize implementation. MemoMark remains in V3 Production Quality And
Delivery, with TX-001, BP-001, and reliability certification closure ahead of
any Expression Style source work.

## 2026-06-22

MemoMark entered V2 Research Phase.

Development, UI polishing, and renderer polishing were paused so the project can become a local-first Photo Presentation Engine with research, specification, and Layout Engine as first-class assets.

The Project Constitution was added as the highest-level repository rule. It clarified that old documentation should not be migrated immediately; new research documentation must be built first, and migration should wait until research specifications stabilize.
